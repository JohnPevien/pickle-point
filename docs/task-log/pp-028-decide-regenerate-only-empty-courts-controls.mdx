## Status

## Goal

Decide whether regenerate-only-empty-courts controls are still needed after fair rotation and manual adjustment controls.

## Acceptance Criteria

- Review current Open Play generation and manual adjustment behavior.
- Decide whether the control is still useful, redundant, or harmful.
- Document the decision in `docs/decisions-log.mdx` if it affects future implementation.
- Add implementation work only if the decision says the control is still needed.

## Validation

- `./node_modules/.bin/vitest run convex/openPlaySessions.test.ts -t "fills only courts without active matches"`: passed.
- Current backend behavior fills only unoccupied courts and excludes players already in active matches.

## Decision

No separate regenerate-only-empty-courts control is needed for MVP. The existing Generate Matches action already behaves as the proposed control.
