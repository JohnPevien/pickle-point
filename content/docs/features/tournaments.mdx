## MVP Entrants

Tournaments use fixed doubles teams.

Each team contains two registered player records. Team composition is set before bracket generation.

## Formats

- Single elimination
- Double elimination
- Round robin

## Lifecycle

- Draft
- Registration Open
- Registration Closed
- Bracket Generated
- Live
- Completed
- Cancelled

## Deferred

- Singles tournaments
- Random partner assignment
- Spin-the-wheel social pairings
