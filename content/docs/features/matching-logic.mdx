## Match Rating

The app converts DUPR and manual skill levels into one internal `matchRating`.

| Skill source | Match rating |
| --- | --- |
| DUPR | Use the DUPR rating |
| Beginner | 2.25 |
| Novice | 2.75 |
| Low Intermediate | 3.25 |
| High Intermediate | 3.75 |
| Advanced | 4.25 |

## Default Matching

MVP open play uses skill-aware balanced matching.

The generator should:

1. Prefer players who have waited longest or played fewer games.
2. Generate possible doubles combinations.
3. Balance team ratings.
4. Penalize recent repeated partners.
5. Penalize exact rematches.
6. Randomly choose among the best valid options when scores are close.

## Repeat Prevention

The rule is not "never repeat." The rule is:

> Avoid repeated partners and exact rematches when enough valid alternatives exist.

Small player pools relax the repeat penalties automatically.

## Stored History

Each completed match should store:

- Session
- Date
- Court
- Team A player IDs
- Team B player IDs
- Winner
- Scores
