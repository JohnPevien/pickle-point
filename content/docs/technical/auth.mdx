## Game Masters

Game Masters need authenticated accounts for workspace and venue management.

## Players

Player login accounts are not required for MVP participation.

Players are represented by registered player records first. Guest and walk-in player records can remain accountless while still joining open plays and tournaments. Account claiming and cross-workspace identity sync are future features.

## Provider Direction

Pickle Point uses WorkOS AuthKit with Convex JWT validation for MVP Game Master authentication.

AuthKit routes:

- `/sign-in` starts hosted sign-in.
- `/sign-up` starts hosted sign-up.
- `/callback` handles the WorkOS OAuth callback.

Admin tenant routes are protected with WorkOS `withAuth` before rendering. Convex receives WorkOS access tokens through a route-backed `ConvexProviderWithAuth` bridge, so backend functions can use `ctx.auth.getUserIdentity()` and map Game Masters by `identity.tokenIdentifier`.
